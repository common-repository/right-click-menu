== Description ==

User will get a site menu by mouse right click on any page.
Admin can set a menu from admin menu section.


== Installation ==

1. Upload `right-click-menu.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
